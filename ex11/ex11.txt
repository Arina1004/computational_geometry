Итерационный метод построения выпуклой оболочки
  Строим начальную оболочку (Берем любые три точки и образуем треугольник, сортируем их по часовой стрелке)
  На каждом шаге пытаемся добавить точку s
    берем какой-то метод для локализации точки
    если точка внутри то идем дальше
    если точка снаружи
      проводоим из нее два луча, которые должны быть касательными к текущей оболочке
        удаляеи точки между лучами и на их место ставим точку s



Для того чтобы провести два луча из точки s к оболочке
  Надо найти близжайшую точку оболочки
    Для левого луча
      От точки идти по часовой стрелке и найти точку,
      через которую проведя луч все точки  выпуклой оболчки будут лежать с одной стороны
    Для правого луча
      От точки идти против часовой стрелке и найти точку,
      через которую проведя луч все точки  выпуклой оболчки будут лежать с одной стороны

Можно взять разные методы для локализации
